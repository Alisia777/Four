# Финансист — ДИ

## Миссия роли
Финансовая прозрачность и безопасность: ДДС, план-факт, контроль расходов, раннее предупреждение кассовых рисков.

## KPI
- Кассовые разрывы: 0
- ДДС план-факт: точность
- P&L план-факт: закрытие месяца по срокам
- Контроль расходов: отклонения объяснены/согласованы

## Зоны ответственности (R)
- Платёжный календарь (7–14 дней вперёд минимум)
- Контроль ДДС: остатки, план, обязательные платежи
- P&L: выручка/себес/реклама/логистика/прочее
- Контроль рекламных расходов совместно с РОП (DRR/ROMI)
- Подготовка согласований на платежи и изменения бюджетов

## Ритм
### Еженедельно (пятница)
- **Weekly финансы**: ДДС факт + план 7–14 дней + риски + решения

### Ежемесячно
- **Monthly финансы**: P&L план-факт + выводы + предложения

## Артефакты
- Платёжный календарь
- Реестр обязательных платежей
- Реестр согласований (цена/реклама/закуп/крупные платежи)

## Согласования
- Любые изменения бюджета/расходов/крупные оплаты → через Опердир (A)
