# ОМ МойСклад — ДИ

## Миссия роли
Чистый учёт и предсказуемые остатки: чтобы продажи/закуп/финансы работали на реальных цифрах.

## KPI
- Ошибки остатков / расхождения — минимум
- Себестоимость внесена вовремя (SLA)
- Скорость обработки приходов/перемещений
- Корректные отгрузки/комплектации

## Зоны ответственности (R)
- Приходы в МойСклад: заведение, проверка, документы
- Себестоимость: внесение/контроль корректности
- Перемещения между складами / подготовка к отгрузке
- Инвентаризации: регулярные сверки, фиксация расхождений
- Статусы остатков для продаж/закупа

## Ритм
### Ежедневно
- Принять приходы/перемещения
- Сверить критичные SKU (риски OOS)

### Еженедельно
- **Weekly МойСклад**: приходы + себестоимость + расхождения + блокеры

## Артефакты
- Журнал расхождений
- Регламент инвентаризации (частота/шаги)
- SLA “себестоимость внесена в течение X часов после прихода”
