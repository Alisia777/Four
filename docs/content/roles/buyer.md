# Закупщик — ДИ

## Миссия роли
Обеспечить товар в наличии в нужном объёме и сроке, по согласованной себестоимости и без кассовых провалов.

## KPI
- OOS по ключевым SKU (0–минимум)
- Сроки поставок (ETA соблюдён)
- Себестоимость (в коридоре)
- Прозрачность статусов (воронка закупа заполнена)

## Зоны ответственности (R)
- План закупа по потребности (по данным продаж/остатков)
- Поставщики: выбор/переговоры/счета/договорённости
- Контроль производства/упаковки/отгрузки/логистики
- Статусы закупок: “что где” + риски + решения
- Подготовка оплат (согласование с финансистом/опердиром)

## Ритм
### Ежедневно
- Обновить статусы “в пути/на производстве/ожидает оплаты”
- Поднять блокеры (сроки, брак, задержки)

### Еженедельно (пятница)
- **Weekly закуп**: воронка + платежи 7–14 дней + риски

## Артефакты
- Воронка закупа (единая таблица)
- План оплат (связка с платёжным календарём)
- Реестр рисков поставок (и решения)

## Согласования
- Любая оплата/крупные суммы → через Финансиста + Опердир (по правилам)
