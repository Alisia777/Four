1.1. Операционный директор
Цель роли: управляемая прибыль и предсказуемая операционка: план-факт, контроль расходов, отсутствие “пожаров”.
Зоны ответственности (A):
1.	P&L/маржа/план-факт по кабинетам/брендам (совместно с финансистом). 
2.	Ритм управления: weekly action plan, контроль исполнения задач в “Трекинг задач”. 
3.	Решения по цене и рекламе: любые изменения цен и увеличение рекламных бюджетов — через согласование (теперь это вы как A). 
4.	Стыки отделов: продукт → закуп → склад/МойСклад → продажи → финансы.
Обязанности по периодам:
•	Ежедневно: 1) принять daily-отчёты; 2) закрыть блокеры; 3) утвердить 
•	Еженедельно: 1) action plan; 2) контроль ДДС/ОПиУ; 3) разбор рекламы/остатков/срыва закупов. 
•	Ежемесячно: закрытие план-факта + корректировка правил матрицы и процессов. 
Артефакты (что обязано появляться “из ваших рук”):
•	“Control Tower” (дашборд-панель руководителя)
•	Weekly Action Plan (таблица на неделю)
•	Реестр согласований: цены/реклама/крупные закупы (прозрачность)
KPI роли:
•	Маржа/прибыль план-факт
•	DRR/ROMI в норме
•	Out-of-stock по топ-SKU ↓
•	Дисциплина отчётности (без пропусков)
