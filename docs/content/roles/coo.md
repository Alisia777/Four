# Опердир (COO) — должностная инструкция

## Миссия роли
Управляемая прибыль и предсказуемая операционка: план-факт, контроль расходов, отсутствие “пожаров”, дисциплина исполнения.

## Цели на 1 месяц (первый цикл)
- Запустить **Control Tower (BI минимум)**: продажи/маржа/DRR/OOS/касса.
- Внедрить **ритм управления** (daily/weekly/monthly) и отчётность без пропусков.
- Зафиксировать **оргструктуру + RACI** и “кто за что отвечает”.
- Включить **реестр согласований**: цена/реклама/закуп/крупные платежи.
- Привести ДИ и регламенты к единому формату (1 страница “кратко” + детали).

## KPI роли
- Маржа ₽ (план-факт), DRR/ROMI в норме
- Out-of-stock по топ-SKU → минимум
- Кассовые разрывы → 0 (контроль 7–14 дней)
- Дисциплина отчётности → 100%
- Скорость решений (цена/реклама/закуп) → в SLA

## Зоны ответственности (A)
- P&L / маржа / бюджет расходов по кабинетам/брендам (совм. с финансистом)
- Цена и реклама: утверждение изменений (через реестр)
- Запасы и закуп: приоритеты, предотвращение OOS
- Ритм управления и контроль исполнения задач

## Ритм управления
### Ежедневно
- Принять daily-отчёты (до 20:00)
- Закрыть блокеры (что мешает плану)
- Утвердить 1–3 ключевых решения дня (цена/реклама/закуп)

### Еженедельно
- **ПН:** Weekly Action Plan (план недели + KPI)
- **ПТ:** разбор недели (продажи/реклама/остатки/касса) + решения на следующую

### Ежемесячно
- План-факт, корректировка правил/матриц/процессов

## Права на решения
- Утверждает: изменения цены, перераспределение рекламных бюджетов, запуск закупа/оплаты.
- Блокирует: любые действия без цифр (маржа/касса/DRR).
