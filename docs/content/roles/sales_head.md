# Руководитель отдела продаж (РОП) — ДИ

## Миссия роли
Стабильный рост продаж и маржи на WB при контроле DRR, без просадок по остаткам и без “хаоса” в рекламе/цене.

## KPI (основные)
- Выручка/заказы/выкупы — план-факт
- Маржа ₽ / маржинальность — план-факт
- DRR / ROMI — в коридоре
- OOS по топ-SKU — минимум
- Дисциплина отчётов (Daily/Weekly) — 100%

## Зоны ответственности (R)
- План продаж (неделя/месяц): цели, декомпозиция, контроль выполнения
- Реклама WB: структура кампаний, ставки, минусация, тесты гипотез
- Цена/скидки: предложения по изменению + расчёт эффекта (согласование через Опердир)
- Карточки/SEO/контент: постановка задач дизайнеру/контенту, контроль внедрения
- Отзывы/вопросы: системная отработка, сбор обратной связи для продукта
- Конкуренты: мониторинг 3–5 ключевых игроков по топ-SKU

## Ритм управления
### Ежедневно (до 20:00)
- Сдать **Daily WB**: продажи/реклама/остатки/действия/что меняем завтра
- Проверить OOS-риски по топ-SKU
- Зафиксировать 3–7 действий на завтра

### Еженедельно (пятница)
- **Weekly WB**: итоги недели + план следующей
- План поставок / дозаказ по рисковым SKU (совм. с Закупом и ОМ МС)

## Обязательные артефакты
- Daily WB (по шаблону)
- Weekly WB (по шаблону)
- Лист “гипотез”: что тестировали, какой эффект, что оставляем

## Правила согласований
- Любые изменения **цены** и **рост бюджета рекламы** → только через Опердир
- Любые “переезды” карточки/названия/фото → фиксируем в отчёте (что поменяли и зачем)

## Взаимодействия
- Продуктолог: требования к карточке/УТП/сбор обратной связи
- Закупщик: план потребности по SKU, прогнозы
- Финансист: маржа/DRR/расходы, проверка эффективности
- ОМ МС: остатки/перемещения/ошибки учёта
