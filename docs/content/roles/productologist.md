# Продуктолог — ДИ

## Миссия роли
Рост маржи через правильный продукт: матрица SKU, УТП, контент, цены, улучшения по отзывам и конкуренции.

## KPI
- Маржа/юнит-экономика по топ-SKU (до/после рекламы)
- Конверсия карточки (CTR/CR — если есть данные)
- Рейтинг/отзывы, доля негатива по ключевым причинам
- Скорость вывода улучшений (SLA по задачам)

## Зоны ответственности (R)
- Матрица товаров: что в рост, что на стоп, что на вывод
- УТП / позиционирование: тексты, офферы, инфографика (ТЗ дизайнеру)
- SEO-ядро: названия/описания/ключи, обновления по сезону
- Анализ конкурентов: цена/контент/комплектация/обещания
- Продуктовые улучшения: “что чинить” по отзывам и возвратам
- Подготовка новинок: от идеи до “готово к закупу”

## Ритм
### Еженедельно
- 3–5 SKU: план улучшений карточки (что меняем, зачем, какой эффект ждём)
- Свод причин негатива по отзывам + решение

### Ежемесячно
- Ревизия матрицы SKU: ABC/маржинальность/скорость оборота

## Артефакты
- Таблица “план улучшений SKU”
- Таблица “конкуренты по топ-SKU”
- Шаблоны ТЗ для дизайна/контента (единый формат)

## Согласования
- Изменение комплектации/сути продукта → через Опердир (и финансы, если влияет на себестоимость)
