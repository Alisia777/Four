# RACI

## RACI (каркас)

**Легенда:**  
- **R** — делает руками (Responsible)  
- **A** — отвечает за результат (Accountable)  
- **C** — консультирует (Consulted)  
- **I** — информируется (Informed)

### Роли
- **COO** — Опердир  
- **РОП** — руководитель продаж  
- **PROD** — продуктолог  
- **PROC** — закупщик  
- **WMS** — ОМ МойСклад  
- **FIN** — финансист  
- **AST** — ассистент  
- **OWN** — собственники

### Матрица процессов (минимально достаточная)

| Процесс | A | R | C | I |
|---|---|---|---|---|
| План продаж / KPI по брендам | COO | РОП | FIN | OWN |
| Управление рекламой (ставки, DRR/ROMI) | COO | РОП | FIN, PROD | OWN |
| Цены / скидки / СПП (решение) | COO | РОП | FIN | OWN |
| Контент / карточки / SEO | РОП | PROD | COO | — |
| Ассортимент / запуск SKU | COO | PROD | PROC, FIN | OWN |
| Закупки / производство / сроки | COO | PROC | WMS, FIN | OWN |
| Приходы / перемещения / корректность учёта | COO | WMS | PROC, FIN | — |
| ДДС (неделя) | FIN | FIN | COO | OWN |
| P&L (месяц) | COO | FIN | РОП | OWN |
| Инциденты / «пожары» | COO | Владелец процесса | FIN/РОП/… | OWN |

> Замена «A = собственник» на **A = COO** — обязательна, иначе система превращается в хаос.
