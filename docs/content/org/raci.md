# RACI (каркас)

**A** — accountable (ответственный за результат)  
**R** — делает  
**C** — консультирует  
**I** — информируется

| Процесс | A | R | C | I |
|---|---|---|---|---|
| Новинка (поиск → unit → “к закупу”) | Продуктолог | Продуктолог | Финансист | Опердир |
| Заказ партии / оплата / сроки | Опердир | Закупщик | Продуктолог, Финансист | ОМ МС |
| Учёт МойСклад (себестоимость/перемещения/инвент) | ОМ МС | ОМ МС | Финансист | Опердир |
