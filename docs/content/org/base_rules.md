# Базовые правила системы

- **Задача = только в трекере задач.** Если в чате — “это не задача”.
- **Дедлайны обязательны.**
- Ежедневно до **20:00** — отчёт “что сделано” (в ваш чат/форму).
- Рабочий контур: **10:00–18:00**, на связи **9:00–20:00**.
- Созвоны: **ПН 11:00** и **ПТ 17:00**.
- Любые изменения **цены** и увеличение **рекламных бюджетов** — только через согласование с **Опердиром (тобой)**.
